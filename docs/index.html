cronlog

Log events. Get an email report later.

This project is parked for now (indefinitely?), due to lack of interest.

Tired of all those emails that get kicked off by those cron jobs? (you know you've got em!) Well, this is a simple service to help clean up your inbox. With a simple curl command, you can consolidate all of that info, and get a single email about it later.

Get an API Key!

Examples

Send your data with curl.

curl
  --data 'message=SOME-MESSAGE&command=SOME-COMMAND&key=API-KEY'
  https://cronlog.io/cronlog/

Or, with Python + requests:

>>> import requests
>>> payload = {
    'command': 'requests.post',
    'message': 'Testing POST w/ requests',
    'host':'localhost',
    'key': 'YOUR-API-KEY'
}
>>> requests.post('https://cronlog.io/cronlog/', payload)

Or with JavaScript + axios:

axios.post('https://cronlog.io/cronlog/', {
    command: 'axios.post',
    message: 'Testing w/ JavaScript'
    host: 'localhost'
    key: 'YOUR-API-KEY'
  })
  .then(function (response) {
    console.log(response);
  })
